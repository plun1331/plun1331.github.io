What is HypixelBot?

HypixelBot is a Discord bot built to display Hypixel player statistics.

HypixelBot Commands

h!help | Displays a list of commands.

h!player | Displays information on a player.

h!bedwars | Displays a player's Bedwars statistics.

h!skywars | Displays a player's Skywars statistics.
